  • 1. Creer les comptes sociaux avec le branding
  • 2. Commencer a poster du contenu (fermes, recettes) SANS mentionner l'app

  • 3. Envoyer a la liste d'attente

08 — Croissance

Strategie influenceurs & codes referral

Comment faire grandir l'app sans budget publicitaire au debut

💡 Pourquoi cette strategie fonctionne

🎯
Audience ultra-cibleeUn micro-influenceur fitness/sante de 2K abonnes a une audience PLUS engagee qu'un compte de 100K generaliste.
💸
Cout quasi zeroLes micro-influenceurs acceptent souvent un code referral + commission plutot qu'un cachet. Pas besoin de budget.
📈
Effet boule de neige50 influenceurs x 2K abonnes = 100K personnes atteintes. Meme 2% de conversion = 2,000 nouveaux users.

📊 Les 3 tiers d'influenceurs a cibler

N

Nano-influenceurs

1K — 10K abonnes

✓ Engagement tres eleve (5-15%)
✓ Audience locale Quebec
✓ Acceptent souvent gratuit
✓ Authentiques et credibles
Niches a cibler : Fitness QC, nutrition, alimentation saine, recettes, CrossFit, running, yoga
Objectif : 50-100 partenaires
M

Micro-influenceurs

10K — 100K abonnes

✓ Bon ratio portee/engagement
✓ Creneau bien defini
✓ Commission ou echange produit
→ Ex: Xavier Lemay (20K-30K)
Exemples de profils : Entraineurs perso QC, dieteticiens, chefs cuisiniers, agriculteurs, blogueurs sante
Objectif : 10-20 partenaires cles
X

Macro-influenceurs

100K+ abonnes

✓ Portee massive
✓ Credibilite instantanee
~ Cachet potentiel requis
→ Ex: Olivier Primault
Approche : Commencer par le code referral, offrir une part des revenus, ou un partenariat officiel avec l'app
Phase 2 (quand l'app tourne)

🔗 Comment le code referral fonctionne

1
Influenceur s'inscrit
On lui cree un code unique
ex: XAVIER10
2
Il partage son code
Bio Instagram, Story, TikTok
Lien de telechargement
3
Ses abonnes signent up
Entrent le code a l'inscription
ou cliquent son lien
4
Il gagne une commission
Ex: 2$ par inscription
ou % si abonnement premium

Ce que l'influenceur gagne :

  • → Commission par inscription (ex: 1-2$ par user)
  • → % des abonnements premium generes
  • → Badge "Partenaire officiel" dans l'app
  • → Profil mis en avant dans la section Recettes

Ce qu'on gagne :

  • → Nouveaux utilisateurs qualifies
  • → Contenu cree gratuitement par les influenceurs
  • → Credibilite dans la communaute sante QC
  • → Croissance organique durable

Phase 2 — Publicites payantes

Quand on a du capital et une base d'utilisateurs

TikTok Ads

Meilleur ROI pour le mobile. Ciblage par interet (fitness, sante, alimentation). Video ads courtes.

Budget min: ~500$/mois
Instagram Ads

Stories + Reels. Retargeting des visiteurs du site. Lookalike audience des premiers users.

Budget min: ~300$/mois
Facebook Ads

Ciblage 30-55 ans, interet agriculture locale, bio, sante. Forte audience Quebec sur Facebook.

Budget min: ~300$/mois
Strategie : D'abord croissance organique (influenceurs + codes referral) → quand on genere des revenus → reinvestir en ads pour accelerer.
